## 🎯 项目背景

### 学习目标
本项目是一个综合性的Java Web开发实践项目，旨在通过开发一个类似飞书的企业级即时通讯系统，学习和掌握现代Web开发的核心技术和最佳实践。

### 技术学习重点
- **微服务架构**: 学习Spring Cloud微服务开发
- **分布式系统**: 理解分布式系统的设计原理
- **实时通信**: 掌握WebSocket实时通信技术
- **数据库设计**: 学习多数据库架构设计
- **前后端分离**: 掌握RESTful API设计
- **容器化部署**: 学习Docker容器化技术

## 🏗️ 技术架构

### 整体架构图

```
┌─────────────────────────────────────────────────────────────┐
│                        前端层                                │
│              Vue.js + Element Plus                          │
└────────────────────┬────────────────────────────────────────┘
                     │ HTTP/WebSocket
┌────────────────────┴────────────────────────────────────────┐
│                      网关层                                  │
│              Spring Cloud Gateway                           │
└────────────────────┬────────────────────────────────────────┘
                     │
┌────────────────────┴────────────────────────────────────────┐
│                    微服务层                                  │
├─────────────────────────────────────────────────────────────┤
│ 用户服务 │ 消息服务 │ 群组服务 │ 文件服务 │ 音视频服务      │
│ (8001)  │ (8002)  │ (8003)  │ (8005)  │ (8008)         │
└────────────────────┬────────────────────────────────────────┘
                     │
┌────────────────────┴────────────────────────────────────────┐
│                    数据层                                    │
├─────────────────────────────────────────────────────────────┤
│ PostgreSQL │ MongoDB │ Redis │ RabbitMQ │ MinIO           │
│ (用户数据) │(消息数据)│(缓存) │(消息队列)│(文件存储)        │
└─────────────────────────────────────────────────────────────┘
```
